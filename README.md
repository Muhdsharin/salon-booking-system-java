# Salon Booking System (Java)

##Overview
A Java-based Salon Booking System developed using Object-Oriented Programming (OOP) and Java Swing GUI.

This system allows customers to book appointments and enables administrators to manage customers, appointments, and staff efficiently.

---

##Features

###Customer
- Book appointment
- Enter personal details
- Choose service, date, and time

###Admin
- View appointments
- Delete customers and their appointments
- Add staff
- Delete staff
- View staff list

---

##Technologies Used
- Java
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
